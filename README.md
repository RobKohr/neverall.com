## install instructions

Requirements

- Mongodb
- Node.js - latest stable

* npm install
* npm run server < --- in one terminal
* npm start < -- in a secondary terminal

Go to app at http://localhost:4000
and click on feedback (the other apps have broken stylesheets)

Click on register to create an account, then log in.

When you are logged in, you will get some more drop down headers, and also if you make the site mobile width, you can see how the menu can slide in from the side.

Most of the app isn't implemented, but you can also go to create and see the form for that, but the backend isn't wired up for that yet.
